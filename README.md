# Vinheria Agnello

Esse projeto se trata de um Sistema básico de cadastro e análise de vinhos artesanais da Vinheira Agnello, proposto pelo professor Lucas Sousa desenvolvido como continuação do Checkpoint 1.

Nesse projeto, foram implementadas as funcionalidades de cadastrar vários vinhos em sequência, verificação de estoque e também a classificação dos vinhos.

## Funcionalidades

* Cadastro de vários vinhos em sequência
* Validação de entradas (evita campos vazios)
* Verificação de estoque baixo (menos de 5 unidades)
* Classificação dos vinhos por safra (jovem, amadurecido ou antigo)
* Exibição dos dados no console
